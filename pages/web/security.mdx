# Computer Security

## Tools

- [dirsearch](https://github.com/maurosoria/dirsearch) is a simple command line
  tool designed to brute force directories and files in websites.

- [Common User Passwords Profiler (CUPP)](https://github.com/Mebus/cupp)

- [Burp suite](https://portswigger.net/burp#essential-manual-tools)

- [Nmap](https://www.explainhownow.com/2019/scan-networks-with-nmap/)

- [Sublist3r](https://github.com/aboul3la/Sublist3r), fast subdomains
  enumeration tool for penetration testers.
