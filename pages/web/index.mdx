## WebExtensions API

- [Awesome WebExtensions](https://github.com/bfred-it/Awesome-WebExtensions) has
  a healthy collection of resources. Of all the links listed, my favorites would
  be:

  - [web-ext](https://github.com/mozilla/web-ext), Mozilla's CLI tool for add-on
    development,
  - [generator-web-extension](https://github.com/webextension-toolbox/generator-web-extension),
    a Yeoman generator to get your add-on started,
  - [webextension-toolbox](https://github.com/webextension-toolbox/webextension-toolbox),
    a small CLI toolbox for creating cross-browser Web Extensions, and
  - [wext-shipit](https://github.com/LinusU/wext-shipit), another CLI tool to
    automate the process of shipping Web Extensions for Chrome, Firefox, Safari,
    Opera and Edge.
