Accessing pageQuery props

After going through the tests below, I settled on the following solution:

// Text.jsx
import React from 'react';
export default props => <>{props.text}</>;

// Paragraph.jsx
import React from 'react';
export default props => <p>{props.text}</p>;

Not sure if this optimal but it produces the desired result:
